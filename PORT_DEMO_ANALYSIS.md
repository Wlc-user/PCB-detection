## 三、推荐Demo方向（按优先级）

### 3.1 Demo 1: 人员安全监控 (最容易落地)

**核心功能**：
```
├── PPE检测 (安全帽、工作服、手套)
├── 区域入侵检测 (禁区闯入)
├── 违规行为检测 (跌倒、打电话)
├── 人数统计 (区域人数)
└── 火焰烟雾检测
```

**商业价值**：
- 事故率降低 60%
- 巡检人员减少 40%
- 客户付费意愿高

**技术方案**：
```
摄像头 ──→ 边缘盒子 ──→ AI检测 ──→ 告警
              │                    │
              ▼                    ▼
         存储录像            微信/短信通知
```

---

### 3.2 Demo 2: 集装箱箱号识别

**核心功能**：
```
├── 箱号OCR识别 (字符识别)
├── 箱型分类 (20ft/40ft/45ft)
├── 残损检测 (凹陷、锈蚀)
├── 箱位定位 (贝位、层位)
└── 数据对接TOS系统
```

**商业价值**：
- 识别准确率 > 98%
- 替代人工录入
- 对接码头系统

---

### 3.3 Demo 3: 起重机状态监测

**核心功能**：
```
├── 吊具状态检测 (双箱/单箱/异常)
├── 钢丝绳安全检测 (断丝/磨损)
├── 集卡定位检测 (对位/不对位)
├── 安全区域检测 (吊臂下方禁入)
└── 作业效率统计
```

**商业价值**：
- 减少事故 70%
- 提前预警故障
- 提高装卸效率

## 六、技术选型建议

### 6.1 模型选择

| 场景 | 推荐模型 | 原因 |
|------|---------|------|
| 通用检测 | **YOLOv10** | 速度快，精度高 |
| 轻量部署 | **YOLOv10n** | 模型最小 |
| 高精度 | **YOLOv10m** | 平衡之选 |
| 分类任务 | **ResNet/EfficientNet** | 成熟稳定 |

### 6.2 硬件选择

| 方案 | 设备 | 价格 | 适用场景 |
|------|------|------|----------|
| 入门 | Jetson Nano | ¥1000 | 1-2路视频 |
| 主流 | Jetson Xavier NX | ¥2500 | 4-8路视频 |
| 高端 | Jetson AGX Orin | ¥8000 | 8-16路视频 |
| 服务器 | RTX 4090 | ¥15000 | 中心推理 |

### 6.3 软件架构

```
├── 推理框架: ONNX Runtime / TensorRT
├── 视频流: FFmpeg / RTSP
├── Web界面: Flask / FastAPI
├── 数据库: SQLite / PostgreSQL
└── 告警: WebSocket / MQTT
```
