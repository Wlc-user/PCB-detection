XCP标定 & ISO 26262功能安全

XCP 协议概述

XCP (Universal Calibration Protocol) - ASAM标准，用于ECU标定、测量、Flash编程

特性说明
协议标准ASAM MC-5, ISO 11791
传输层CAN/CAN FD/Ethernet/USB/SPI
连接模式Master(测试仪) → Slave(ECU)
主要功能读/写内存、标定参数、DAQ测量

XCP 命令集

// 连接命令
0xF0: CONNECT      // 建立连接
0xF1: DISCONNECT   // 断开连接
0xF2: GET_STATUS   // 获取状态
0xF3: GET_SEED     // 请求种子
0xF4: UNLOCK       // 解锁

// 上传/下载
0xF4: UPLOAD        // 读取内存
0xF5: SHORT_UPLOAD  // 短读取
0xE0: DOWNLOAD      // 写入内存
0xE1: DOWNLOAD_NEXT // 下一块
0xE2: DOWNLOAD_MAX  // 最大块下载
// DAQ命令
0xC0: GET_DAQ_LIST_SIZE    // DAQ列表大小
0xC1: ALLOC_DAQ           // 分配DAQ
0xC2: FREE_DAQ            // 释放DAQ
0xC3: ALLOC_ODT           // 分配ODT
0xC4: ALLOC_ODT_ENTRY     // 分配ODT条目
0xE4: START_STOP_DAQ_LIST // 启动/停止
0xE5: START_STOP_SYNC      // 同步启动
0xC6: WRITE_DAQ           // 写DAQ配置

// 校准
0xF7: BUILD_CHKSUM        // 校验和
0xF8: CLEAR_DAQ          // 清除DAQ

XCP 通信示例

CONNECT
建立连接
→
SET_MTA
设置地址
→
UPLOAD
读取数据
→
DOWNLOAD
写入校准
// CONNECT
TX: F0 00              // CONNECT, mode=0 (normal)
RX: FF 01 0B 00 00    // RES, Resource=0x01(DAQ+STIM)
                         // CTO size=11, DTO size=0

// SET_MTA (设置内存地址)
TX: F3 00 00 00 00 10 00 00
// RESERVED, EXT=0, addr=0x00100000

// UPLOAD (读取8字节)
TX: F4 08              // UPLOAD, size=8
RX: FF 11 22 33 44 55 66 77 88  // 读取的数据

// DOWNLOAD (写入校准)
TX: E4 08 AA BB CC DD EE FF 00 11
RX: FF                  // 成功响应

DAQ 数据采集

DAQ (Data Acquisition) 用于实时采集ECU内部信号，用于标定和验证 

// DAQ 配置流程
// 1. 分配DAQ列表
TX: C0 00              // GET_DAQ_LIST_SIZE
RX: FF 00 0A          // 10个DAQ列表可用

TX: C1 00 0A          // ALLOC_DAQ, count=10
RX: FF

// 2. 分配ODT和条目
TX: C3 00 00 03       // DAQ=0, ODT=0, count=3
TX: C4 00 00 00 00 00 01 // DAQ=0, ODT=0, idx=0, addr=?, size=1

// 3. 配置DAQ
TX: E4 00              // START_STOP, mode=0 (stop)
TX: E4 01              // mode=1 (start once)
TX: E4 02              // mode=2 (continuous)
