## 四、产品功能

### 4.1 缺陷检测

检测6类PCB常见缺陷：

| 缺陷类型 | 英文名 | 危险等级 |
|---------|--------|----------|
| 缺失孔洞 | missing_hole | 🔴 极高 |
| 鼠咬缺陷 | mouse_bite | 🟡 中 |
| 开路 | open_circuit | 🔴 高 |
| 短路 | short | 🔴 极高 |
| 毛刺 | spur | 🟡 中 |
| 多余铜 | spurious_copper | 🟢 低 |

### 4.2 根因定位

基于DSSM双塔匹配算法，分析缺陷根因，反推异常工序：

```
输入: 缺陷图片 → 特征提取 → 双塔匹配 → 输出: 可能工序
```

**支持工序**：
- 表面处理：清洗、沉铜、蚀刻、阻焊
- 组装：SMT贴片、回流焊、波峰焊
- 检测：AOI、X-Ray、ICT

### 4.3 自动化能力

```
全流程自动化:
│
├─ 自动触发 (光电传感器/PLC)
├─ 自动拍照 (工业相机)
├─ 自动检测 (AI推理)
├─ 自动判定 (OK/NG)
├─ 自动记录 (数据库)
├─ 自动输出 (PLC信号)
└─ 无人值守 (7×24小时)
```
