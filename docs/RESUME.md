## 项目经历

### 🏆 AutoTest Academy - 汽车电子学习平台 ⭐

**GitHub**: https://github.com/Wlc-user/AutoTest-Academy

**项目简介**: 面向汽车电子工程师的在线学习平台，涵盖CAPL编程、CAN/CAN FD通信、UDS诊断协议、Bootloader等核心技术栈。

**技术栈**: HTML5 / CSS3 / JavaScript / CANoe / CAPL / UDS / CAN FD

**核心模块**:
- **CAPL基础教程**: CANoe脚本编程入门，包含交互式代码示例
- **CAPL测试框架**: 自动化测试用例设计 + 在线评测系统(OJ)
- **UDS状态机**: ISO 14229统一诊断服务协议详解与模拟器
- **CAN FD通信**: 车载网络协议实战演练
- **Bootloader开发**: OTA远程升级原理与实现
- **XCP标定**: ECU参数标定技术
- **ODX诊断**: 开放诊断数据交换格式
- **车载安全**: SecOC/HSM/入侵检测

**项目亮点**:
- 纯前端实现，无需后端，支持离线访问
- 响应式设计，适配桌面/移动端
- GitHub Stars 增长中，开源社区认可

### 🚗 YOLO PCB缺陷检测系统

**技术栈**: Python / PyTorch / YOLOv8 / OpenCV / TensorRT

**核心工作**:
- 基于YOLOv8的PCB电路板缺陷检测模型训练
- 支持 6 类缺陷: 缺孔、短路、毛刺、开路、余铜、鼠洞
- TensorRT INT8量化优化，推理速度提升 3x
- ONNX模型导出，支持多平台部署

**模型性能**:
