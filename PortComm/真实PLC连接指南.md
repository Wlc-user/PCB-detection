# PortAI 真实PLC连接指南 v4.0

## 连接方式

### 方式1: Modbus TCP (以太网)

**硬件连接:**
```
电脑 ←→ 交换机 ←→ PLC
   (网线)    (网线)
```

**配置步骤:**
1. 确保电脑和PLC在同一网络
2. 记下PLC的IP地址 (如: 192.168.1.100)
3. 确认PLC的Modbus端口 (默认: 502)

**HMI设置:**
- ✅ 勾选 **TCP**
- IP: `192.168.1.100` (改成实际PLC IP)
- Port: `502`

**点击 "连接"**


### 方式2: Modbus RTU (串口/RS485)

**硬件连接:**
```
电脑 ←→ USB转RS485 ←→ PLC
   (USB线)   (信号线)
```

**常见接线:**
| USB-RS485 | PLC |
|-----------|-----|
| A (D+) | A (D+) |
| B (D-) | B (D-) |
| GND | GND |

**HMI设置:**
- ✅ 勾选 **Serial**
- COM: `COM3` (在设备管理器查看实际端口)
- Baud: `9600` (根据PLC配置)

**点击 "连接"**


## TX/RX 指示灯

| 指示灯 | 含义 | 正常状态 |
|--------|------|----------|
| TX (蓝色) | 发送数据 | 连接后闪烁 |
| RX (绿色) | 接收数据 | PLC响应时闪烁 |
| 状态 ● | 连接状态 | 绿色=已连接 |

**TX/RX正常闪烁 = 通信正常!**


## 故障排除

| 问题 | 解决方法 |
|------|----------|
| 连接超时 | 检查IP/端口是否正确 |
| 端口被占用 | 关闭其他程序或换端口 |
| TX闪但RX不闪 | PLC未响应，检查接线 |
| 数据全是0 | 检查寄存器地址 |
| CRC错误 | 确认串口参数一致 |


## 推荐Modbus PLC品牌

- **西门子 S7-200/S7-1200** (需要Modbus库)
- **三菱 FX5U** (内置Modbus TCP/RTU)
- **施耐德 PLC** (原生Modbus)
- **汇川 H5U** (支持Modbus)
- **国产PLC** (大多支持Modbus)
