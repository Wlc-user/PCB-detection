## 真实PLC连接配置

### 1. Modbus TCP 连接 (推荐)

支持: 西门子 S7-1200 + Modbus TCP模块, 施耐德 PLC, 三菱 Q系列

```python
# config.py 配置
"tcp": {
    "enabled": True,
    "host": "192.168.1.100",  # PLC IP地址
    "port": 502,               # 标准Modbus TCP端口
    "timeout": 5
}
```

### 2. Modbus RTU (串口) 连接

支持: RS485转USB, 变频器, 传感器

```python
"serial": {
    "enabled": True,
    "port": "COM3",       # Windows串口号
    "baudrate": 9600,
    "bytesize": 8,
    "parity": "N",        # N=无校验, E=偶校验, O=奇校验
    "stopbits": 1
}
```

### 3. Siemens S7 连接 (需要 snap7)

```bash
pip install snap7
```

## 寄存器地址映射

根据真实PLC配置地址:

| 地址 | 名称 | 真实设备 | 说明 |
|------|------|----------|------|
| 0 | CRANE_ST | 起重机PLC | 0=停止, 1=运行, 2=故障 |
| 1 | CONT_POS | 位置编码器 | 0-100% |
| 2 | TRUCK_CNT | 地磁传感器 | 车辆计数 |
| 3 | SHIP_DET | 雷达/摄像头 | 0/1 |
| 4 | ALARM | 安全系统 | 告警标志 |
| 5 | SPEED | GPS/RFID | km/h |
| 6 | TEMP | 温湿度传感器 | °C |
| 7 | WEIGHT | 地磅 | kg |

## 启动步骤

### 快速启动 (模拟模式)
```bash
python PLCSimulator.py    # 启动模拟PLC
python production_hmi.py   # 启动上位机
```

### 真实PLC连接
```bash
# 1. 确认PLC IP
ping 192.168.1.100

# 2. 测试Modbus连接
telnet 192.168.1.100 502

# 3. 启动上位机
python production_hmi.py
```

## 数据导出

日志自动记录到 CSV 文件:
- 路径: `data_log.csv`
- 格式: 时间戳, 8个寄存器值, 告警状态

## 故障排查

### 连接失败 (10061)
```bash
# 检查PLC是否在线
ping 192.168.1.100

# 检查端口是否开放
telnet 192.168.1.100 502
```

### 读取超时
```python
# 增加超时时间
"timeout": 10  # 秒
```

### 数据不正确
- 检查寄存器地址映射
- 确认数据类型 (int16/int32/float)
- 验证字节序 (big/little endian)
