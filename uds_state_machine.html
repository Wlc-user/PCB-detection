🔧 常用诊断服务

SID服务功能
0x10DiagnosticSessionControl会话控制
0x11ECUResetECU复位
0x22ReadDataByIdentifier按DID读数据
0x27SecurityAccess安全访问
0x2EWriteDataByIdentifier按DID写数据
0x19ReadDTCInformation读取故障码
0x14ClearDTC清除故障码
0x34/36/37RequestDownload/Transfer/Exit刷写流程

📦 报文结构

// 正响应 = SID + 0x40 // 负响应 = 0x7F + Rej-SID + NRC // 示例: 切换到扩展会话 TX: 10 03 // 请求 RX: 50 03 00 32 00 C3 // P2=50ms, P2*=5000ms // 安全访问流程 TX: 27 01 // 请求种子 RX: 67 01 5A A5 C3 3C // 返回4字节种子 TX: 27 02 A5 5A 3C C3 // 发送密钥

⚠️ 常见否定响应码 (NRC)

NRC名称含义
0x11serviceNotSupported服务不支持
0x12subFunctionNotSupported子功能不支持
0x22conditionsNotCorrect条件不满足
0x33securityAccessDenied安全访问被拒绝
0x35invalidKey密钥无效
0x78requestCorrectlyReceived等待中(ECU忙)
