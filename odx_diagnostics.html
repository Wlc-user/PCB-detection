📊 ODX 诊断数据库

什么是 ODX?

ODX (Open Diagnostic Data Exchange) - ISO 22901-1 定义的诊断数据交换标准，用于统一管理ECU诊断数据

ODX编辑器
→
CANoe
→
诊断仪

ODX 文件结构

// PDX 包文件结构
PDX (Package)
├── COMPARAMS      // 通信参数
│   ├── CAN_params
│   ├── LIN_params
│   └── Eth_params
├── DIAG-SERVICES  // 诊断服务定义
│   ├── SERVICE_0x10
│   ├── SERVICE_0x27
│   └── SERVICE_0x19
├── DIAG-COMM-SPEC // 服务通信规范
├── DTC-DESCRIPTIONS // DTC定义
├── VEHICLE-INFO    // 车辆信息
└── ECU-VARIANTS    // ECU变体

// 文件扩展名
*.odx  // 单个诊断数据文件
*.pdx  // 包文件(多个odx)
*.mdx  // 制造商特定格式

ODX XML 示例

<?xml version="1.0" encoding="UTF-8"?>
<ODX MODEL-VERSION="2.2.0">
  
  <DIAG-SERVICE ID="SVC_0x1003">
    <SHORT-NAME>ExtendedSession</SHORT-NAME>
    <LONG-NAME>Extended Diagnostic Session</LONG-NAME>
    <SID>0x10</SID>
    
    <SUBFUNCTION REQ-BYTE="0x03"/>
    
    <POS-RESPONSE SPEC-ID="POS_RESP_01">
      <DID-REFS>
        <DID ID="DID_P2MAX"/>
        <DID ID="DID_P2STAR"/>
      </DID-REFS>
    </POS-RESPONSE>
    
    <NEG-RESPONSE-REFS>
      <NEG-RESPONSE-ID-REF ID-REF="NRC_0x12"/>
      <NEG-RESPONSE-ID-REF ID-REF="NRC_0x22"/>
    </NEG-RESPONSE-REFS>
    
  </DIAG-SERVICE>

  <COMPARAM-SPEC ID="CAN_SPEC">
    <COMPARAM SUBTYPE="CAN-PARAMS">
      <PARAM NAME="BITRATE">500000</PARAM>
      <PARAM NAME="SAMPLE_POINT">87.5</PARAM>
    </COMPARAM>
  </COMPARAM-SPEC>
  
</ODX>
